The following information is about the structures of all the files.               

functions - All the MATLAB functions

Please firstly run the script cw1_data.m to geterate the data, then you can run the script cw1.m to get the result figures.
The script in cw1_data.m may take long time to runn. In this case, you can directly use the provided data cw1.mat to run the script in cw1.m to get the result figures.
